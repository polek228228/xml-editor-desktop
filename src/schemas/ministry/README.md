## Использование

```javascript
// Валидация XML против XSD
const schemaPath = 'src/schemas/ministry/pz-01.05/explanatorynote-01-05.xsd';
const xmlValidator = new XMLValidator(schemaPath);
const isValid = xmlValidator.validate(xmlDocument);
```
