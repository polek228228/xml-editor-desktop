📦 ВАРИАНТ 1: PORTABLE ZIP (ГОТОВ СЕЙЧАС) ✅

4. Скрипт автоматически:
   → Проверяет Node.js (или предлагает установить)
   → Устанавливает зависимости (3-10 минут)
   → Создает ярлык на рабочем столе
   → Запускает приложение

ПЛЮСЫ:
✅ Готов прямо сейчас
✅ Не требует прав администратора
✅ Работает с USB-флешки
✅ Легко обновлять

МИНУСЫ:
❌ Требует Node.js на машине пользователя
❌ Первая установка занимает 3-10 минут

🚀 ВАРИАНТ 2: .EXE INSTALLER через GitHub Actions (5-10 минут)

НАСТРОЕНО: .github/workflows/build-windows.yml

КАК ЗАПУСТИТЬ:

Способ А - Ручной запуск (рекомендуется для теста):
---------------------------------------------------------
1. Запушить код на GitHub:
   git add .
   git commit -m "feat: Add GitHub Actions for Windows build"
   git push origin main

2. Открыть GitHub → Actions

3. Выбрать "Build Windows Installer"

4. Нажать "Run workflow" → "Run workflow"

5. Подождать 5-10 минут

6. Скачать windows-installer.zip из секции "Artifacts"

7. Внутри будет XMLEditorDesktop-Setup-1.0.0.exe (~110-150 MB)


Способ Б - Автоматическая сборка при релизе:
---------------------------------------------------------
1. Создать тег:
   git tag v1.0.0
   git push origin v1.0.0

2. GitHub автоматически:
   → Соберет .exe
   → Создаст Release
   → .exe будет на странице Releases


ЧТО ПОЛУЧИТСЯ:
✅ XMLEditorDesktop-Setup-1.0.0.exe (~110-150 MB)
✅ Полноценный NSIS installer
✅ Графический мастер установки
✅ Не требует Node.js
✅ Регистрация в системе
✅ Удаление через Control Panel
✅ Все зависимости включены


СТОИМОСТЬ:
✅ БЕСПЛАТНО (публичные репозитории)
✅ 2000 минут/месяц (приватные)


ПОДРОБНАЯ ИНСТРУКЦИЯ:
