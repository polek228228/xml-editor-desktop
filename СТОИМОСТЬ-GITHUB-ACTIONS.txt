⏱️  СКОЛЬКО ЗАНИМАЕТ ОДНА СБОРКА?

Windows .exe installer:  ~10 минут (600 секунд)

Из них:
  • 2 минуты - установка Node.js
  • 3 минуты - npm ci (установка зависимостей)
  • 5 минут - npm run build:win (сборка .exe)

❓ ЧАСТЫЕ ВОПРОСЫ:

Q: Нужно ли платить за GitHub Actions?
A: НЕТ, если:
   • Репозиторий публичный (unlimited бесплатно)
   • Репозиторий приватный + собираете < 200 раз/месяц

Q: Что если превышу лимит 2000 минут?
A: Сборки остановятся до следующего месяца
   Или можно купить дополнительные минуты ($0.008/мин)
   Или купить GitHub Pro ($4/мес, получить 3000 минут)

Q: Можно ли собирать бесплатно навсегда?
A: ДА! Сделайте репозиторий публичным → unlimited бесплатно

Q: Что быстрее - GitHub Actions или собирать локально на Windows?
A: Примерно одинаково (~10 минут)
   Но GitHub Actions не требует Windows машины!

Q: Можно ли собирать одновременно несколько версий?
A: ДА! GitHub Actions поддерживает параллельные сборки
   (но каждая считается отдельно)
