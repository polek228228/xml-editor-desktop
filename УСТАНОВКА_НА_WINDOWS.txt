  КАК УСТАНОВИТЬ XML Editor Desktop НА WINDOWS
═══════════════════════════════════════════════════════════

📦 ДВА ВАРИАНТА УСТАНОВКИ:
──────────────────────────────────────────────────────────

1. ПОЛНАЯ УСТАНОВКА (рекомендуется)
   Файл: XML Editor Desktop-Setup-1.0.0.exe

2. ПОРТАТИВНАЯ ВЕРСИЯ (без установки)
   Файл: XML Editor Desktop-Portable-1.0.0.exe


═══════════════════════════════════════════════════════════
  ВАРИАНТ 1: ПОЛНАЯ УСТАНОВКА
═══════════════════════════════════════════════════════════

ШАГ 1: ЗАПУСТИТЬ УСТАНОВЩИК
──────────────────────────────────────────────────────────
Дважды кликните на:
  XML Editor Desktop-Setup-1.0.0.exe


ШАГ 2: РАЗРЕШИТЬ WINDOWS DEFENDER
──────────────────────────────────────────────────────────
Если Windows Defender блокирует:
1. Нажмите "Подробнее"
2. Нажмите "Выполнить в любом случае"

Это нормально для неподписанных приложений!


ШАГ 3: СЛЕДОВАТЬ ИНСТРУКЦИЯМ УСТАНОВЩИКА
──────────────────────────────────────────────────────────
1. Выберите папку установки (по умолчанию: C:\Program Files\)
2. Создать ярлык на рабочем столе: ДА
3. Создать ярлык в меню Пуск: ДА
4. Запустить после установки: ДА


ШАГ 4: РАЗРЕШИТЬ ИЗМЕНЕНИЯ
──────────────────────────────────────────────────────────
Windows может спросить:
"Разрешить приложению вносить изменения?"
→ Нажмите ДА


ШАГ 5: ГОТОВО!
──────────────────────────────────────────────────────────
Приложение запустится автоматически.


═══════════════════════════════════════════════════════════
  ВАРИАНТ 2: ПОРТАТИВНАЯ ВЕРСИЯ
═══════════════════════════════════════════════════════════

ШАГ 1: РАСПАКОВАТЬ
──────────────────────────────────────────────────────────
1. Запустите XML Editor Desktop-Portable-1.0.0.exe
2. Выберите папку для распаковки (например: D:\Programs\)
3. Дождитесь завершения


ШАГ 2: ЗАПУСТИТЬ
──────────────────────────────────────────────────────────
Зайдите в папку и запустите:
  XML Editor Desktop.exe


ПРЕИМУЩЕСТВА ПОРТАТИВНОЙ ВЕРСИИ:
- Не требует установки
- Можно запускать с флешки
- Не создаёт записи в реестре


═══════════════════════════════════════════════════════════
  ❌ ЕСЛИ НЕ ЗАПУСКАЕТСЯ
═══════════════════════════════════════════════════════════

ОШИБКА: "Отсутствует VCRUNTIME140.dll"
──────────────────────────────────────────────────────────

РЕШЕНИЕ:
Приложение требует Visual C++ Redistributable 2015-2022.

1. Скачайте установщик:
   https://aka.ms/vs/17/release/vc_redist.x64.exe

2. Запустите установщик

3. Перезагрузите компьютер

4. Запустите XML Editor Desktop снова


ОШИБКА: "Приложение не может запуститься"
──────────────────────────────────────────────────────────

РЕШЕНИЕ:
1. Убедитесь что у вас Windows 10 или 11
2. Убедитесь что установлен Visual C++ Redistributable
3. Попробуйте запустить от имени администратора:
   Правый клик → Запуск от имени администратора


═══════════════════════════════════════════════════════════
  📍 ГДЕ ХРАНЯТСЯ ДАННЫЕ
═══════════════════════════════════════════════════════════

ПОЛНАЯ УСТАНОВКА:
Программа: C:\Program Files\XML Editor Desktop\
Данные: C:\Users\ВАШ_ПОЛЬЗОВАТЕЛЬ\AppData\Roaming\xml-editor-desktop\

ПОРТАТИВНАЯ ВЕРСИЯ:
Всё в одной папке где распаковали


База данных (документы):
C:\Users\ВАШ_ПОЛЬЗОВАТЕЛЬ\AppData\Roaming\xml-editor-desktop\xmleditor.db


═══════════════════════════════════════════════════════════
  🗑️  УДАЛЕНИЕ
═══════════════════════════════════════════════════════════

ПОЛНАЯ УСТАНОВКА:
1. Панель управления → Программы и компоненты
2. Найдите "XML Editor Desktop"
3. Нажмите "Удалить"

ПОРТАТИВНАЯ ВЕРСИЯ:
Просто удалите папку с программой


УДАЛЕНИЕ ДАННЫХ (опционально):
Удалите папку:
C:\Users\ВАШ_ПОЛЬЗОВАТЕЛЬ\AppData\Roaming\xml-editor-desktop\


═══════════════════════════════════════════════════════════
  ℹ️  СИСТЕМНЫЕ ТРЕБОВАНИЯ
═══════════════════════════════════════════════════════════

- Windows 10 или Windows 11 (64-bit)
- 4 GB RAM (рекомендуется 8 GB)
- 500 MB свободного места на диске
- Visual C++ Redistributable 2015-2022 (x64)
