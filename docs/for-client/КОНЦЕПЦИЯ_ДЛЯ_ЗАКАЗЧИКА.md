## 🤔 Зачем это нужно? (проблематика)

### Сегодня вы работаете с xmlonline.ru. Отличный сервис! Но есть нюансы:

**1. Зависимость от интернета**
- Пропал интернет → работа встала
- Медленный интернет → приложение тормозит
- На объекте часто нет стабильного Wi-Fi

**2. Безопасность данных**
- Все ваши проекты хранятся на чужом сервере
- Конкурентная проектная документация уходит в облако
- Нет гарантии, что данные не утекут

**3. Нет истории изменений**
- Изменили раздел 3 вчера → сегодня хотите вернуться к старой версии → **невозможно**
- Нет возможности посмотреть "что было изменено 2 недели назад"
- Один неверный клик → данные потеряны

**4. Медленная работа**
- Каждое сохранение → запрос на сервер → задержка 1-2 секунды
- Открытие большого документа → загрузка 5-10 секунд
- Переключение между разделами → каждый раз подгрузка

**5. Платите за всё сразу**
- xmlonline.ru (если бы был платным): 25,000 руб/год за ВСЕ типы документов
- Но вы используете только Пояснительную записку?
- Всё равно платите полную стоимость
