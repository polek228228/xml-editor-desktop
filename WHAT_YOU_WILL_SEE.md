# 👀 ЧТО ТЫ УВИДИШЬ В ПРИЛОЖЕНИИ

## 📝 WYSIWYG Редактор (как Word)

Когда создашь документ и прокрутишь вниз, увидишь поля с панелью инструментов:

```
┌─ Описание проекта ──────────────────────────────────┐
│ [↶ ↷] [Формат▼] [B I U] [≡≡≡≡] [•① ← →] [✗] [⊞] [?] │ ← Панель кнопок
├─────────────────────────────────────────────────────┤
│                                                      │
│  Здесь можно писать текст с форматированием         │
│  **жирный** *курсив* _подчёркнутый_                 │
│  • Списки                                            │
│  1. Нумерованные                                     │
│                                                      │
└─────────────────────────────────────────────────────┘
```

## 📋 Динамические списки (Repeater)

В разделе **"Список экспертов"** увидишь:

```
┌─ Список экспертов ──────────────────────────────────┐
│                                                      │
│  ┌─ #1 ──────────── [↑] [↓] [×] ─────────┐         │ ← Кнопки управления
│  │ ФИО эксперта:   [Иванов И.И.        ] │         │
│  │ Должность:      [Главный инженер    ] │         │
│  │ Телефон:        [+7 999 123-45-67   ] │         │
│  └──────────────────────────────────────┘         │
│                                                      │
│  ┌─ #2 ──────────── [↑] [↓] [×] ─────────┐         │
│  │ ФИО эксперта:   [Петров П.П.        ] │         │
│  │ Должность:      [Архитектор         ] │         │
│  │ Телефон:        [+7 999 765-43-21   ] │         │
│  └──────────────────────────────────────┘         │
│                                                      │
│  [+ Добавить Эксперт]                               │ ← Добавить элемент
│                                                      │
└─────────────────────────────────────────────────────┘
```

**Что делают кнопки:**
- **↑** - Переместить элемент выше
- **↓** - Переместить элемент ниже  
- **×** - Удалить элемент
- **+ Добавить** - Добавить новый элемент

## 🎯 Как проверить:

1. Запусти: `npm run dev`
2. Кликни **"Documents"** (верх)
3. Кликни **"+ Новый документ"** (слева)
4. Заполни: Название = "Тест", Схема = "pz-01.05"
5. **Прокрути вниз** до:
   - "Описание проекта" ← WYSIWYG редактор
   - "Технические решения" ← WYSIWYG редактор
   - "Список экспертов" ← Repeater со стрелками
   - "Примечания" ← WYSIWYG редактор

## 🐛 Если не видишь:

1. Открой DevTools (View → Toggle Developer Tools)
2. Посмотри вкладку Console на ошибки
3. Проверь что TinyMCE загрузился: в консоли должно быть `[RichTextEditor] TinyMCE initialized`
